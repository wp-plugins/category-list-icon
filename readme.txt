Display category icon on list.

== Description ==

add different icon to each category name on category list.

Live Preview: http://kentothemes.com/demo/category-list-lcon/

By this plugin you could add different icon or image in-front of category name on category list.



Plugin Features

* Custom icon size.
* no icon if not save icon image.
* Custom CSS to each category list items.




== Installation ==


1. Install as regular WordPress plugin.
2. Go your Pluings setting via WordPress Dashboard and activate it.
3. Before test plugin you need to add Categories widget on sidebar.
4. After activate plugin you will see menu at left menu bar "CLI Settings" please set value for<br /> 

<strong>Icon Size:</strong>(ex: 20)<br />
 

<strong>How to display category icon ?</strong><br /> 
To display category icon you must save an .png image exactly same as image name of category slug to <strong>"cat-images"</strong> folder or directory on this plugin.

for example if you want to display category icon for "facebook" on your site then you have to save an image  "facebook.png" to <strong>"cat-images"</strong>(inside this plugin, folder alredy exist) folder or directory then it will automatically added before tag name on category list.
